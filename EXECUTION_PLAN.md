## 📞 问题排查

### 问题1: SSH连接失败
```
错误: ssh: connect to host 101.42.21.191 port 22: Connection refused

解决:
1. 检查服务器IP地址是否正确
2. 检查防火墙是否开放22端口
3. 检查本地网络是否正常
4. 尝试 ssh -v root@101.42.21.191 查看详细信息
```

### 问题2: git pull失败
```
错误: fatal: not a git repository

解决:
1. 确保已进入正确目录: cd /www/wwwroot/epgo
2. 检查是否是Git仓库: ls -la | grep .git
3. 如果没有，可能需要重新克隆:
   git clone git@github.com:xiachaoqing/epgo.git /www/wwwroot/epgo
```

### 问题3: 后台无法打开
```
错误: 无法连接到 101.42.21.191/admin/

解决:
1. 检查MetInfo是否正确安装
2. 检查Apache/Nginx是否运行
3. 清除浏览器缓存
4. 检查防火墙是否开放80/443端口
```

### 问题4: 页面显示不正常
```
错误: 首页显示乱码、样式混乱等

解决:
1. 清除MetInfo缓存（已在任务1.4中说明）
2. 清除浏览器缓存: Ctrl+Shift+Delete
3. 检查文件权限: chmod -R 755 /www/wwwroot/epgo
```
