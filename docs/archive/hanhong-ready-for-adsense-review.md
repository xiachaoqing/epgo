```
1. 访问 https://search.google.com/search-console
2. 选择你的网站
3. 在"爬虫统计"中点击"请求编入索引"
4. 输入首页URL并提交
5. 等待Google完整爬取所有新文章（24-48小时）
```

### 步骤2: 提交AdSense申请
```
1. 访问 https://www.google.com/adsense/
2. 点击"注册"或"立即开始"
3. 填写网站信息（已准备好）
4. Google会进行审核（3-5个工作日）
5. 审核通过后即可在网站上展示广告
```

### 步骤3: 等待审核完成
```
1. 初期审核：3-5个工作日
2. 深入审核：额外1-2周
3. 最终决定：通常在2-4周内
```
