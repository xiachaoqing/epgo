### 添加随机阅读量

在"高级选项"或"自定义字段"中，如果MetInfo支持，可以设置：

```
点击量/阅读数: 1200 (对应文章1)
点击量/阅读数: 890  (对应文章2)
点击量/阅读数: 1540 (对应文章3)
... 以此类推
```

**如果MetInfo后台不支持直接编辑阅读量**，可以使用以下方法：

1. 通过数据库直接修改 (需要SSH权限)
   ```sql
   UPDATE ep_news SET hits = 1200 WHERE id = (文章ID);
   ```

2. 或在模板中通过JavaScript生成随机数显示

## ⚠️ 注意事项

### 文章编辑器使用

1. **富文本编辑**: MetInfo支持富文本编辑，可以：
   - 设置标题、加粗、斜体
   - 添加列表、表格、图片
   - 调整颜色和字体

2. **HTML编辑**: 如果需要更复杂的格式，可以切换到HTML编辑模式

3. **SEO字段**: 务必填写：
   - 关键词
   - 摘要/描述
   - URL友好名称

### 文件上传限制

- 最大文件大小: 通常为100MB (可在后台设置)
- 支持格式: PDF, DOC, DOCX, XLSX, ZIP等
- 建议压缩文件以提高上传速度

### 发布后的优化

发布文章后，为了让搜索引擎更快收录：

1. 清除缓存
2. 向Google提交sitemap
3. 验证页面无异常
