## 🔍 故障排除

### 问题1：连接失败 "Can't connect to MySQL"

**原因**: 数据库连接失败
**解决**:
- 检查数据库是否运行
- 检查主机地址是否正确
- 检查用户名和密码是否正确
- 确保在服务器上运行，而不是本地

### 问题2：栏目不存在 "栏目 '关于我们' 不存在"

**原因**: 栏目名称不匹配
**解决**:
- 检查MetInfo后台中栏目的实际名称
- 修改脚本中的栏目名称
- 或手动指定栏目ID

### 问题3：缺少Python库

**错误**: `ModuleNotFoundError: No module named 'pymysql'`
**解决**:
```bash
pip3 install pymysql
# 或
pip install pymysql
```

### 问题4：权限不足

**错误**: `Permission denied`
**解决**:
```bash
chmod +x fill_content.py
python3 fill_content.py
```
