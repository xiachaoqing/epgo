## 部署步骤

### 1. 本地准备
```bash
# 确保所有文件已修改和测试
# 创建备份
cp -r templates/epgo-education templates/epgo-education.backup
```

### 2. 文件上传
```bash
# 通过 FTP/SFTP 上传以下文件夹
# templates/epgo-education/
# 确保权限正确（644 for files, 755 for directories）
```

### 3. MetInfo 操作
1. 登录后台管理
2. 系统设置 > 清除缓存
3. 静态文件 > 清除缓存
4. 日志 > 检查错误日志

### 4. 验证部署
1. 清除浏览器缓存（Ctrl+Shift+Delete）
2. 按 F5 或 Ctrl+R 刷新页面
3. 检查开发者工具（F12）
   - Console: 无错误
   - Network: 文件加载成功
   - Elements: HTML 结构正确

## 回滚计划

如果出现问题：

```bash
# 恢复备份
rm -rf templates/epgo-education
mv templates/epgo-education.backup templates/epgo-education

# MetInfo 清除缓存
# 重新登录后台，清除所有缓存

# 等待 5-10 分钟后重新测试
```

## 常见问题排查

### 问题：样式没有生效
**解决方案：**
1. 清除浏览器缓存（Ctrl+Shift+Delete）
2. MetInfo 后台清除模板缓存
3. 确认 config.json 正确
4. 检查 CSS 文件是否上传成功

### 问题：动画卡顿
**解决方案：**
1. 检查浏览器版本（需要支持 CSS Transform）
2. 禁用浏览器扩展
3. 检查系统资源占用
4. 在移动设备上测试

### 问题：二维码不显示
**解决方案：**
1. 检查 MetInfo 后台配置
2. 确认二维码文件存在
3. 检查文件权限（644）
4. 尝试重新上传

### 问题：页面加载缓慢
**解决方案：**
1. 检查服务器连接速度
2. 压缩大图片文件
3. 启用 Gzip 压缩
4. 使用 CDN 加速（如配置）
