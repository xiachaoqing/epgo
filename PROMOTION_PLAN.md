## 🎯 推广策略（分步执行）

### 【第一步】网站与APP的连接

**目标：** 让网站访问者知道有APP，且能方便下载

**方案 A：首页顶部 Banner**
```
位置：首页最上方或醒目位置
内容：
  "🎁 限时特惠：下载APP，享受更丰富的学习资源 + 专业指导"
  按钮：[下载APP] → https://app.lingshi.com/bjxxsy
```

**方案 B：侧边栏/悬浮窗**
```
固定在右侧或浮动在底部
显示APP特点，吸引点击
```

**方案 C：文章底部 CTA**
```
每篇文章下方添加：
  "想要系统学习？下载APP，获得完整课程 + 真人语音讲解"
```

### 【第二步】微信公众号引流

**目标：** 将微信粉丝导向网站和APP

**方案：**
1. **菜单栏添加**
   - "免费学习" → 链接到网站首页
   - "下载APP" → APP下载链接

2. **文章推广**
   - 每周发布：网站热门文章截取
   - 底部添加：完整版在网站/APP查看

3. **公众号图文卡片**
   - 设置图文转链到网站文章

### 【第三步】APP内的反向推广

**目标：** APP用户知道有官方网站

**方案：**
1. APP 首页/菜单添加"官方网站"链接
2. APP 内某些功能提示引导访问网站获取免费资源

### 【第四步】内容差异化

**网站内容策略：**
- 保持免费、基础内容
- 部分文章添加"APP订阅用户可看全部"标签

**APP内容策略：**
- 完整课程、语音讲解、练习题
- 离线下载、个人学习计划
- 实时反馈和成绩统计
